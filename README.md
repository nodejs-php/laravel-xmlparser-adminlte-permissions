# Парсер товаров

Установка docker и запуск сервера
``./vendor/bin/sail up``

Запуску миграции
``./vendor/bin/sail artisan migrate``

Команда выполнения загрузки данных
``sail artisan products-load``

Для получения данных в режиме Excel
``/export``
